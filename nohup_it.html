nohup it

converts any command into a nohup with log




Backgroud: nohup

All commands get nohuped in sequence. Only 1 command per line (newlines in between).

FYI:

  • 'bash -c' is to allow multiline
  • 'set -x' print the command also in output file
  • 's$(date +"%Y_%m_%d_%H_%M_%S")' to append current datetime to the filename. example output file nohup_2018_03_18_18_45_59.log
  • 'less -f $(ls -1t /tmp/nohup* | sed q)' to tail the latest output log file nohup_.....log
