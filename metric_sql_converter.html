Lily Metric to SQL converter

Converts provided metric names to it's SQL form for use in Lily DNA JDBC driver




Backgroud: Metric to SQL logic

The logic for when an underscore is inserted:

  • When going from lowercase to uppercase
  • When going from non-numeric to numeric and when going from numeric to non-numeric.
    This makes that "sumAmountOver5k" translates to "SUM_AMOUNT_OVER_5_K" or that "field1" translates to "FIELD_1"
  • When there is already an underscore, no extra underscores are inserted
